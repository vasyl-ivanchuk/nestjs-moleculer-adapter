## [Nest.js app](https://github.com/nestjs/nest) with an adapter to interact with [moleculer.js services](https://github.com/moleculerjs/moleculer).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
